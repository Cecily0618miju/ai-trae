# 匠心出圈：AI 非遗爆款短视频导演

> 让传统技艺被年轻人看见。把传承人的真实口述和技艺细节，转成今天就能拍的钩子、脚本、分镜与现场清单。

## 项目介绍

「匠心出圈」是一款面向非遗传承人、传统手艺人、地方文旅账号、文化类短视频博主、学校社团和文创团队的 **AI 短视频创作工具**。

它不是普通短视频生成器，也不是带货工具，而是一个 **AI 非遗短视频导演助手**。用户输入非遗项目、技艺特点、传承人故事、目标受众、视频平台和传播目标后，系统自动生成完整短视频执行方案。

### 解决的真实问题

很多非遗传承人有技艺有故事，但不擅长短视频表达，导致内容传播弱、年轻人看不懂、好手艺难以被看见。本产品用 AI 帮非遗内容完成从「传统介绍式展示」到「年轻人愿意看、愿意转发」的转化。

## 核心功能

### 6 大页面

1. **首页** — 六段式产品叙事：定位、前后对比、核心能力、工作流、实时 Demo、案例与 CTA
2. **创作工作台** — 四步采集向导 + 本地草稿 + 逐步校验 + 六阶段生成反馈
3. **AI 生成结果页** — 三栏导演工作台 + 10 大模块完整方案（见下）
4. **示例案例** — 糖画、皮影戏、剪纸、苏绣 4 个内置案例，即点即看
5. **作品库** — 已生成方案列表 + Markdown / TXT / JSON 真实导出
6. **关于项目** — 项目背景、AI 解决流程、社会价值

### 10 大生成模块

| 模块 | 内容 |
|------|------|
| 爆款选题推荐 | 3 个选题，含标题、适合平台、推荐理由、传播亮点 |
| 短视频完整脚本 | 按时间轴展示，每段含画面/旁白/字幕/拍摄建议 |
| 镜头分镜表 | 镜头编号、景别、画面、运镜、字幕、拍摄重点 |
| 封面文案 | 悬念型、情绪型、知识型三种封面 |
| 字幕与口播文案 | 短句、有节奏、年轻化、保留文化温度 |
| 平台适配版本 | 抖音 / 小红书 / 视频号 / B站的封面和开头钩子 A/B 方案 |
| 拍摄清单 | 道具、场景、特写、人物、环境、收音、光线 |
| 方案就绪度 | 5 维度可解释评估 + 逐项评分依据（不做流量承诺） |
| 非遗文化内容核验 | 核心关键词、表达风险、建议保留内容与发布前本人核对提示 |
| AI 优化建议 | 3 条针对性优化建议 |

结果页首屏额外展示「真实素材锚点」，直接回显用户技艺细节和传承人原话，便于评委验证“输入真的影响输出”。

### 动态生成逻辑

输入「糖画」生成糖画方案，输入「剪纸」生成剪纸方案。生成引擎覆盖糖画、剪纸、皮影戏、苏绣、木版年画、泥塑、蓝印花布等关键词，其他输入有通用非遗模板兜底，确保任何输入都有完整输出。

## 比赛亮点

### 1. 创新性

- **不是普通 AI 视频工具**：聚焦非遗传播场景，突出「AI 导演」「3 秒钩子」「分镜设计」「拍摄就绪度」
- **真正懂非遗**：内置糖画、剪纸、皮影、苏绣等专属模板，生成内容贴合具体技艺特点
- **AI 全链路**：从选题到评分，覆盖短视频创作完整流程，而非单一文案生成

### 2. 实用性

- **完整体验链路**：从输入信息到生成方案全过程可体验，不停留在概念介绍
- **真实使用场景**：面向非遗传承人、文旅账号、博主等明确目标用户
- **可直接执行**：分镜表与拍摄清单像真正能拿去拍摄的方案，不空泛

### 3. 完成度

- **核心链路跑通**：四步采集 → 六阶段生成 → 10 大模块结果 → 作品库，全流程无断点
- **6 个页面完整**：首页、工作台、结果、案例、作品库、关于，无空页面
- **所有按钮可点击**：生成、保存、导出、查看案例全部联通

### 4. 美观度

- **非遗文化感**：米白/朱砂红/鎏金/墨黑/青绿色调，云纹/印章/纸张纹理装饰
- **年轻化**：现代卡片布局、圆角阴影、半透明叠加，非传统政府网站风格
- **专业动效**：Framer Motion 实现页面入场、状态反馈与分镜联动，并兼容系统的“减少动态效果”设置

## 评委演示路径（2 分钟）

> 评委可按以下路径快速验证完整链路，全程无需注册、无需联网。建议提前 `npm run dev` 启动开发服务器。

### 第 0 秒 ｜ 首页：创新点前置展示
- 首屏 Hero 区：「匠心出圈」定位 + 立即生成方案 CTA
- 下滑到「**AI 导演团工作流**」：六步可视化（文化识别 → 爆款选题 → 分镜导演 → 平台适配 → 内容守护 → 方案评估）
- 下滑到「**实时 Demo**」：点击「运行 AI 导演团」，观察输入「糖画」后选题、分镜、评分三段输出依次生成（约 2 秒）

### 第 30 秒 ｜ 创作工作台：生成自己的方案
- 点击首页「免费生成导演方案」进入工作台
- 按四步向导填写项目、真实技艺与人物原话（也可点击「糖画示例」）
- 选择发布平台、目标受众、视频时长与传播目标，点击「生成导演方案」
- 观察六位 AI 导演的待处理 / 处理中 / 已完成状态

### 第 60 秒 ｜ 结果页：查看 10 大模块（重点看创新点）
- **模块三 · 导演分镜卡**：含景别、运镜、AI 画面提示词、构图建议、时长，点击分镜卡联动右侧视频预览器
- **模块六 · 平台适配版本**：每个平台有 **A/B 封面标题对比** + **3 秒开头钩子 A/B 对比**，AI 推荐钩子有高亮标记
- **模块八 · 方案就绪度**：5 维雷达图 + 每个维度的「**评分依据**」，明确说明这是拍摄准备度而非流量承诺
- **模块九 · 非遗文化内容核验**：非遗识别、核心技艺关键词、表达风险、建议保留内容与发布前核对提示

### 第 90 秒 ｜ 导出与作品库
- 点击「保存方案」→ 弹出 Toast「已保存到作品库」
- 点击「导出拍摄包」→ 可真实下载拍摄清单、分镜脚本、Markdown 或 JSON
- 顶部导航进入「作品库」→ 看到刚保存的方案卡片，可点击查看完整方案

### 第 120 秒 ｜ 示例案例（可选加餐）
- 顶部导航进入「示例案例」→ 糖画 / 皮影戏 / 剪纸 / 苏绣 4 个内置案例即点即看
- 任选一个进入结果页，验证不同非遗项目的差异化生成效果

## Demo 说明

- 本 Demo 使用前端 mock 数据，**无需联网**，无需登录注册
- AI 生成逻辑为前端本地规则引擎模拟，用户的技艺细节与传承人原话会进入脚本、首镜和导出包
- 传播相关分数仅用于评估素材/方案就绪度，不预测或承诺真实流量
- 内置糖画完整案例可直接查看
- 4 个示例案例（糖画/皮影/剪纸/苏绣）可即点即看
- 导出功能可真实下载 Markdown / JSON，也可复制完整方案
- 完全响应式，手机和电脑均可正常浏览
- 可直接构建为静态网站部署
